douban-client

douban-client 是对豆瓣 API v2 接口进行了一个简单封装，主要包括了 OAuth 2.0 认证、图片上传以及接口方面的调用。

目前已完成的接口有：

* 用户 People
* 广播 Miniblog
* 豆邮 Doumail
* 日记 Note
* 相册 Album
* 图片 Photo
* 线上活动 Online
* 讨论区 Discussion

正在进行开发中的接口有:

* 豆瓣猜 Guess
* 同城活动 Event
* 音乐 Music
* 电影 Movie
* 图书 Book
* 回复 Comment

安装

pip install douban-client

或

easy_install douban-client

使用说明

通用接口:

获取 client.xxx.get()
新建 client.xxx.new()
更新 client.xxx.update()
删除 client.xxx.delete()
列表 client.xxx.list()

默认参数（参考豆瓣官方文档）:

# 在需要传 start, count 的接口，默认：
start: 0
count: 20
＃ 如需更改，请自定指定

用户 People

# 以下 id 部分指用户数字 id
获取当前用户信息 client.people.me
获取指定用户信息 client.people.get(id) 
搜索用户    client.people.search(q)       # q: 搜索的关键词

关注用户    client.people.follow(id)
取消关注    client.people.unfollow(id)
加入黑名单  client.people.block(id)
获取粉丝信息  client.people.followers(id, start, count) # 谁关注了此 id
获取关注者信息 client.people.following(id, start, count) # 此 id 关注了谁
查看关注关系  client.people.friendships(target_id, source_id) # 如不指定 source_id 则使用当前用户
查看共同关注  client.people.follow_in_common(id, start,count) # 查看当前用户与指定 id 共同关注好友

已实现的接口中单元测试覆盖超过 90%，如果文档中有没有说明的可以参考下： https://github.com/liluo/douban-client/tree/master/tests
